# Cards v0 Balance Analysis

本文件分析第一批 10 张测试卡牌的平衡考量。所有计算基于默认 d6 骰子。

## 基础假设

- 默认骰子: d6 (点数 1~6，平均 3.5)
- 玩家基础配置: 30 HP, 12 Energy, 2d6 每回合
- 测试敌人: TrainingBeast (20 HP, 固定攻击 14)

## 卡牌平衡分析

### 1. EnergyStrike

| 指标 | 值 |
|------|-----|
| Energy Cost | 1 |
| Dice Cost | 1 |
| 伤害公式 | dice + 2 |
| 最小伤害 | 3 |
| 最大伤害 | 8 |
| 平均伤害 | 5.5 |
| 伤害/Energy | 5.5 |

**分析**: 基础攻击卡，性价比合理。平均每 Energy 造成 5.5 伤害，适合前期清理。

### 2. BreakCore

| 指标 | 值 |
|------|-----|
| Energy Cost | 3 |
| Dice Cost | 1 |
| 基础伤害 | 8 |
| 破甲触发条件 | dice >= 5 (33%概率) |
| 破甲加成期望 | 2 × 0.33 = 0.66 |
| 总伤害期望 | 8 + 0.66 = 8.66 |
| 伤害/Energy | 2.89 |

**分析**: 高消耗卡，基础伤害稳定但性价比偏低。破甲是延迟收益，需要配合后续攻击才能体现价值。

### 3. QuickStrike

| 指标 | 值 |
|------|-----|
| Energy Cost | 0 |
| Dice Cost | 0 |
| 伤害 | 4 |
| 伤害/Energy | ∞ |

**分析**: 零消耗卡，作为免费伤害来源。在 Energy 不足时仍可打出，保证每回合最低输出。

### 4. DoubleStrike

| 指标 | 值 |
|------|-----|
| Energy Cost | 2 |
| Dice Cost | 2 |
| 伤害公式 | floor((dice1 + dice2) × 1.5) |
| 最小伤害 | 3 |
| 最大伤害 | 18 |
| 平均伤害 | 10.5 |
| 伤害/Energy | 5.25 |

**分析**: 双骰子卡，方差较大。平均伤害略低于 EnergyStrike，但上限更高。需要预留 2 枚骰子才能打出。

### 5. DefensivePosture

| 指标 | 值 |
|------|-----|
| Energy Cost | 1 |
| Dice Cost | 0 |
| Energy Recovery | 3 |
| 净收益 | +2 Energy |

**分析**: 能量恢复卡，净赚 2 Energy。适合在低骰子回合使用，或为高消耗卡积累能量。

### 6. DiceBoost

| 指标 | 值 |
|------|-----|
| Energy Cost | 2 |
| Dice Cost | 0 |
| 效果 | +1 枚骰子 |

**分析**: 骰子生成卡，用 2 Energy 换 1 枚骰子。适合配合多骰子卡使用，提高爆发潜力。

### 7. PowerStrike

| 指标 | 值 |
|------|-----|
| Energy Cost | 4 |
| Dice Cost | 2 |
| 基础伤害 | (dice1 + dice2) + 6 |
| 额外伤害条件 | dice_sum >= 9 (27.78%概率) |
| 额外伤害期望 | 5 × 0.2778 = 1.39 |
| 基础平均伤害 | 13 |
| 总伤害期望 | 14.39 |
| 伤害/Energy | 3.60 |

**分析**: 高消耗高风险卡。基础伤害不错，但额外伤害触发门槛较高。需要充足能量和骰子。

### 8. VulnerableStrike

| 指标 | 值 |
|------|-----|
| Energy Cost | 2 |
| Dice Cost | 1 |
| 伤害公式 | dice + 1 |
| 破甲触发条件 | dice >= 3 (67%概率) |
| 破甲加成期望 | 1 × 0.67 = 0.67 |
| 基础平均伤害 | 4.5 |
| 总伤害期望 | 5.17 |
| 伤害/Energy | 2.59 |

**分析**: 中等消耗卡，破甲触发概率高。基础伤害偏低，但破甲收益稳定，适合持续输出。

### 9. EnergySurge

| 指标 | 值 |
|------|-----|
| Energy Cost | 0 |
| Dice Cost | 1 |
| 恢复公式 | dice + 1 |
| 最小恢复 | 2 |
| 最大恢复 | 7 |
| 平均恢复 | 4.5 |

**分析**: 零能量卡，用骰子换能量。平均恢复 4.5 Energy，适合在有多余骰子时使用。

### 10. CriticalHit

| 指标 | 值 |
|------|-----|
| Energy Cost | 2 |
| Dice Cost | 1 |
| 伤害公式 | dice + 3 |
| 暴击条件 | dice >= 4 (50%概率) |
| 非暴击平均伤害 | 5 |
| 暴击平均伤害 | 16 |
| 总伤害期望 | 10.5 |
| 伤害/Energy | 5.25 |

**分析**: 高风险高回报卡。50%概率暴击，伤害翻倍。期望伤害不错，但方差较大。

## 配合与策略

### 前期爆发 (1~2 回合)
- EnergyStrike × 2 + QuickStrike: 期望 5.5 + 5.5 + 4 = 15 伤害
- 适合快速压低敌人血量

### 破甲流
- VulnerableStrike → EnergyStrike → EnergyStrike: 破甲后后续攻击伤害+1~2
- 适合持续战斗

### 高爆发流
- DiceBoost + DoubleStrike/PowerStrike: 额外骰子提高上限
- 适合清理残血敌人

### 能量管理
- DefensivePosture + EnergySurge: 积累能量为高消耗卡做准备
- 适合应对敌人高攻击回合

## 潜在平衡问题

### 问题 1: QuickStrike 零消耗是否过强？
- 免费 4 伤害，无任何代价
- 可能导致玩家每回合保底输出过高
- **建议**: 后续版本可增加限制，如每回合只能用 1 张零消耗卡

### 问题 2: BreakCore 性价比偏低
- 3 Energy 仅造成 8 伤害
- 破甲收益延迟，前期可能卡手
- **建议**: 可考虑降低 Energy Cost 至 2，或增加基础伤害

### 问题 3: 多骰子卡门槛较高
- DoubleStrike 和 PowerStrike 需要 2 枚骰子
- 默认只有 2d6，打出后可能无骰子可用
- **建议**: 配合 DiceBoost 使用，或增加初始骰子数量

## 与测试敌人的互动

### TrainingBeast (20 HP, 攻击 14)
- 理想回合数: 2~3 回合
- 第一回合: EnergyStrike + EnergyStrike + QuickStrike ≈ 15 伤害
- 第二回合: 剩余 5 HP，任意攻击卡均可击杀

### TrainingDummy (999 HP)
- 适合测试持续输出和状态累积
- 破甲流在 TrainingDummy 上效果明显

## 后续调整方向

1. **增加新状态**: 如燃烧、中毒、虚弱等，丰富策略维度
2. **调整消耗曲线**: 增加更多 0~1 Energy 消耗的卡牌
3. **优化多骰子卡**: 考虑增加单骰子高上限的卡牌
4. **增加手牌管理**: 如弃牌、抽牌机制

## 验证标准

在训练场中验证以下场景：

- [ ] 第一回合手牌 3 张，能打出所有类型的卡
- [ ] EnergyStrike 平均伤害接近 5.5
- [ ] BreakCore 在 dice >= 5 时正确施加破甲
- [ ] 破甲在后续攻击中正确增加伤害并减少层数
- [ ] QuickStrike 无需骰子和能量即可打出
- [ ] 多骰子卡在骰子不足时无法打出
